# 🎵 Rhythm Blast - 节奏爆破

一个基于 Web 的下落式节奏游戏，测试你的音游天赋和实力！

## ✨ 功能特色

- 🎮 **4轨道下落式玩法** - 经典音游操作，使用 D F J K 键
- 🎯 **精准判定系统** - PERFECT / GREAT / GOOD / MISS 四级判定
- 🔥 **连击加成** - Combo 越高分数倍率越高
- 🎵 **多首歌曲** - 4种难度，从入门到专家
- ⚡ **天赋测试** - 手速CPS测试 + 反应速度测试
- ✨ **炫酷特效** - 粒子效果、判定动画、动态背景

## 🎹 操作方式

| 按键 | 功能 |
|------|------|
| D | 第1轨道 |
| F | 第2轨道 |
| J | 第3轨道 |
| K | 第4轨道 |
| ESC / 空格 | 暂停游戏 |

## 🚀 快速开始

直接在浏览器中打开 `index.html` 即可游玩！

或者使用本地服务器：

```bash
python -m http.server 8080
# 然后访问 http://localhost:8080
```
